1. Overview

Albion Price Checker is still in the development stage and aims to be a cross-platform utility tool for Albion Online players. It's intended to be the pocket app: an easy to use tool that lets you check item prices in no time. It will also feature a personal regear program so you can save your ZvZ sets, always know where to get them, and how much you have to pay. For leaders and managers out there, a regear program at large scale will also be available with additional features, like crafting cost for regear.

Below is a list of all features that will be included in the application. More features may be included as time goes or as requested:

  • Price Checker
  • Personal Regear Program
  • Guild/Alliance Regear Program
  • Crafting Calculator
  • Account system to support alts and several characters

2. Get Started

Download the application from the download page. This directs to the project's GitHub page's release section. Find the latest version of the application, which is always at the top. Expand Assets and download "albion_price_checker.zip." For the time being, the application is not available to download on Android or iOS. It's only available on Windows.

If you are using the first Development Stage version, then you can run the program with albion-price-checker.exe. Later versions may change this due to compatibility across operating systems.
